# Lapzy — Princípios

## Filosofia do Produto
App de cronometragem para kart focado em performance e uso durante pilotagem.
O piloto tem atenção limitada — cada decisão de design deve respeitar isso.

## Princípios Fundamentais

- **Mínimo de cliques**: iniciar corrida com 1 toque a partir da tela inicial
- **Atenção limitada**: interface legível e operável durante pilotagem
- **Interface limpa**: zero poluição visual, zero distrações
- **Foco em performance**: cada elemento existe para ajudar o piloto a ser mais rápido

## Regras de Design Derivadas

- Se uma ação exige mais de 2 toques, está errada
- Informação crítica (tempo, delta) deve ser legível a 60cm de distância
- Nunca bloquear o início de corrida por qualquer motivo (login, permissão, etc.)
- Feedback visual deve ser imediato e inequívoco (cores fortes, sem ambiguidade)
